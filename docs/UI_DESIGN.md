## 一、配色方案

### 主色调

```scss

$colors: (
  // 主色 - 樱花粉

  success: #A8E6CF,
```

### 玻璃拟态效果

```scss
// 玻璃拟态基础样式
@mixin glass-morphism($opacity: 0.75) {
  background: rgba(255, 255, 255, $opacity);
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.3);
  box-shadow: 0 8px 32px rgba(255, 183, 197, 0.15);
}

// 卡片玻璃效果
@mixin card-glass {
  @include glass-morphism(0.75);
  border-radius: 20px;
  padding: 24px;
}

// 导航栏玻璃效果
@mixin navbar-glass {
  @include glass-morphism(0.85);
  border-radius: 0 0 20px 20px;
  border-top: none;
}
```

```scss
// 过渡时间
$transitions: (
  fast: 0.15s,
  normal: 0.3s,
  slow: 0.5s,
);

// 缓动函数
$easings: (
  ease-in-out: cubic-bezier(0.4, 0, 0.2, 1),

@keyframes fadeIn {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }

@keyframes float {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10px); }

  0% { background-position: -200% 0; }

  0% { transform: translateY(-10vh) rotate(0deg); opacity: 1; }
```

### 4.4 导航栏 (SakuraNavbar)

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  🌸 梦境日记              首页  我的梦境  个人中心    👤    │
│                                                             │
└─────────────────────────────────────────────────────────────┘

样式规格:
- 位置: position: fixed，始终固定在页面顶部
- 高度: 64px
- z-index: 1000

滚动行为:
- 初始状态（页面顶部）: 
  - 背景透明
  - 无阴影效果
- 滚动后状态（scrollY > 50px）:
  - 应用玻璃拟态效果 (opacity: 0.9)
  - 添加阴影: 0 2px 20px rgba(0, 0, 0, 0.1)
  - 平滑过渡: transition: 0.3s

页面适配:
- 在登录/注册页面隐藏导航栏
- 内容页面需添加顶部间距 (padding-top: 80px) 避免被遮挡
```
